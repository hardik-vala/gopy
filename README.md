## Installation

`cd` to the `src` directory and then run the install script:

```
./install.sh
```

## Milestone #2

### How to Build/Run

* Unzip SableCC 3.7 in `./src` directory
* `cd src` and then `make build`
* `java golite.Main -[scan | parse | pretty | pptype| printTokens | dumpsymtab] [filename]`
